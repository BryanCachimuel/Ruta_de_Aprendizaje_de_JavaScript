JavaScript (JS) es un lenguaje de programación que añade interactividad y dinamismo a las páginas web, permitiendo crear contenido que se actualiza sin recargar la página, animaciones, validación de formularios y otras funciones dinámicas

Java es un lenguaje de programación y una plataforma informática multiplataforma y orientada a objetos, creada por Sun Microsystems (ahora parte de Oracle) en 1995. Es conocido por su versatilidad, permitiendo el desarrollo de una amplia gama de aplicaciones, desde software empresarial y aplicaciones móviles hasta juegos y soluciones para el Internet de las Cosas (IoT)

Python es un lenguaje de programación de alto nivel, interpretado y de código abierto, muy fácil de aprender y usar, que se destaca por su sintaxis clara y legible. Es ampliamente utilizado en desarrollo web, ciencia de datos, inteligencia artificial, automatización y desarrollo de software en general, debido a su versatilidad y a la gran cantidad de bibliotecas disponibles

Golang o Go es un lenguaje de programación compilado, de código abierto y desarrollado por Google, diseñado para ser eficiente, fácil de aprender y con un fuerte soporte para la concurrencia. Es muy utilizado en el desarrollo backend, la creación de microservicios, APIs y sistemas distribuidos, gracias a su capacidad para ejecutar múltiples tareas simultáneamente mediante "goroutines".
